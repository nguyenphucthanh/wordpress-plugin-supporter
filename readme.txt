Manage supporters with floating widget

== Description ==

Add floating widget on front page to display supporter (with Skype, Yahoo and Phone...). And this plugin will display widget without manual adding any widget to any sidebar...

== Installation ==

Installing "Supporter" can be done either by searching for "Supporter" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
